HOW TO USE:
===========
1. Create a folder named "images" in your project root
2. Save all product photos with the names listed above
3. Replace "attachment:0", "attachment:1", etc. with actual file paths
4. Update HTML files to use: src="images/filename.jpg"

Example replacement in HTML:
<img src="attachment:0" alt="Women's Formal Pant">
→
<img src="images/womens-formal-pant.jpg" alt="Women's Formal Pant">
